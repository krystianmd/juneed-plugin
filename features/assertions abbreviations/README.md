# Assertion abbreviations

## Introduction
Live templates for commonly used JUnit assertions.

## Available assertions
| Assertion     | Abbreviation  |
| ------------- |:-------------:|
| assertAll     | asa           |
| assertEquals  | ase           |
| assertFalse   | asf           |
| assertTrue    | ast           |
| assertNull    | asn           |
| assertNotNull | asnn          |
| assertThrows  | asth          |
| assertTimeout | asti          |
